🔁 常用控制台指令

  • allowtraits：允許將領自由分配特質（重複但補上說明）
  • manpower：增加 10M 人力（可視需要與現有項目合併）
  • instant_wargoal：瞬間完成正當化
  • allowdiplo：無條件外交，可直接宣戰
  • yesman：AI 接受所有外交提案
  • tp：部隊瞬移（teleport 的簡寫）
  • ic：瞬間建造（等同於 instantconstruction）
  • it：瞬間訓練（等同於 instanttraining）
  • ff：無視國策限制（等同於 focus.nochecks）
  • roic：點擊科技圖示即完成研究（等同於 research_on_icon_click）

🧰 內政外交

  • add_opinion [來源國tag] [目標國tag]：提高或降低兩國間的好感度
  • whitepeace [國tag] [國tag]：雙方立即簽署白和平
  • observe：進入觀察模式，AI 自動控制
  • winwars：立即贏得當前所有戰爭
  • add_party_popularity [意識形態] [數值]：調整政黨支持度（例：d、c、f、n）
  • set_ruling_party [意識形態]：設定執政黨（例：fascism）
  • tag_color [RGB]：更改目前國家顏色（如 tag_color 255 0 0）
  • instantconstruction：建築立即完成
  • allowdiplo：無外交限制，可宣戰
  • Focus.AutoComplete：立刻完成國家方針
  • focus.nochecks：忽略國策前置條件
  • decision.nochecks：忽略決議條件
  • yesman：AI 同意所有外交提案
  • manpower：+100萬人力
  • cp：+100 指揮點
  • ws：設定戰爭支持度
  • st：設定穩定度
  • XP：+500 陸海空經驗
  • PP：+政治點數
  • threat：增加世界緊張度
  • focus.ignoreprerequisites：無視是否完成前置國策
  • civilwar [意識形態] [國家代碼]：發動指定國家內戰

🗺 領土控制

  • tdebug：顯示國家、省份、事件、裝備等 ID（啟用/關閉切換）
  • tag：切換控制國家
  • add_core：增加核心領土
  • annex：吞併國家
  • delall：刪除國家所有軍隊
  • setowner：設為該國擁有
  • setcontroller：設為該國控制

🧪 科技研究

  • research complete：完成第一欄研究
  • research all：完成所有研究
  • research_on_icon_click：點擊完成科技

⚔ 戰爭部隊

  • teleport [省份ID]：將選定部隊瞬間移動到指定省份
  • spawn [單位ID] [省ID] [數量]：在指定地點生成部隊（需啟用 -debug）
  • deleteallunits [國tag]：刪除指定國所有軍隊
  • moveunit [unit ID] [province ID]：將特定單位移動至省份
  • instant_wargoal：立刻完成正當化
  • instanttraining：部隊立即訓練完成
  • instant_prepare：海軍入侵立即準備
  • show_enemy_ships：顯示敵方艦隊
  • bypass_invasion_superiority_check：無視制海權
  • tp：部隊瞬移
  • fow：去除戰爭迷霧，顯示敵方部隊

🎖 將領經驗

  • gain_xp：增加將領經驗

🛢 資源能源

  • fuel：增加燃料

  • ale [數字]：一次性獲得各類最高級裝備（不含艦船）

🏭 工業設計

  • building_health [建築類型] [省/州ID] [等級] [數值]：修改建築耐久度
  • add_ideas [想法ID]：加入指定國策或民族精神
  • IndustrialOrganisation.AddFunds 80000：+8萬設計商資金
  • IndustrialOrganisation.AddFunds 800000：+80萬設計商資金

🧪 模組調試指令

  • spawn：模組開發常用，快速生成單位以測試平衡性或效果
  • reloadtechnologies：修改科技檔後套用（不需重啟）
  • debug_terrain：觀察地形錯誤或地圖開發時使用
  • reloadinterface：修改 UI 檔後重新載入
  • tdebug：取得 province/state ID，用於 add_core、goto_province 等
  • debug_nogui：拍攝純地圖用，關閉所有 UI
  • reloadweather：測試天氣系統模組時使用
  • debug_borders：確認國界顯示是否正確

🧭 使用省份與州 ID 的技巧

  • tdebug 啟用後，滑鼠移到地圖上會顯示 state ID 與 province ID，可搭配 add_core、teleport、goto_province 使用
  • goto_state [ID]：鏡頭移動到指定州
  • goto_province [ID]：鏡頭移動到指定省

🧭 地圖指令圖例

  • 快捷指令提示：
    • add_core [省ID] [國家Tag]：為某州加入核心
    • teleport [省ID]：部隊瞬移
    • setowner [Tag] [省ID]：設為指定國家擁有
    • setcontroller [Tag] [省ID]：設為指定國家控制
    • goto_state [ID]、goto_province [ID]：快速移動鏡頭至指定區域
    • tdebug：顯示省州 ID，與地圖層切換搭配最佳
